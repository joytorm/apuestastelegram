# FRONTEND PLAN (SEO + UX) — ApuestasTelegram

## Objetivo
Lanzar una landing sólida en Next.js con base SEO técnica y UX clara para adquisición y validación de interés.

## Stack frontend elegido
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Metadata API nativa de Next (SEO)
- `context7` instalado para soporte de documentación/consulta
